# Daily Expense CRUD Manager with AI/ML Integration

A comprehensive expense management system that combines traditional CRUD operations with machine learning and deep learning capabilities for intelligent financial insights.

## Features

### Core CRUD Operations
- **Create**: Add new expenses with categories, amounts, and dates
- **Read**: View and filter expenses with pagination
- **Update**: Modify existing expense records
- **Delete**: Remove unwanted expenses

### AI/ML Features
- **Expense Categorization**: Automatic category prediction using ML
- **Spending Pattern Analysis**: Identify trends and anomalies
- **Budget Prediction**: Forecast future expenses using time series analysis
- **Anomaly Detection**: Detect unusual spending patterns
- **Smart Recommendations**: Personalized saving suggestions

## Technology Stack

- **Backend**: Python, Flask
- **Machine Learning**: Scikit-learn, TensorFlow
- **Data Processing**: Pandas, NumPy
- **Visualization**: Matplotlib, Plotly
- **Database**: SQLite with SQLAlchemy

## Installation

1. Clone the repository:
```bash
git clone https://github.com/yourusername/daily-expense-ai-manager.git
cd daily-expense-ai-manager
```

2. Install dependencies
```bash
pip install -r requirements.txt
```

3. Run the application
```bash
python src/app.py
```
